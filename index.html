Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTMl uses id as an attribute
  • h1 - h6 determines the heading tier
  • Semantic HTML elements directly describe its contents and role as opposed to non-semantic HTML elements
  • ul =unlisted, ol =ordered list, li =list items, list items belong within ul or ol sections
  • you can have unlimited list items within the body

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three main ways to style CSS, inline CSS, internal CSS Style Sheet and external CSS style sheet
  • The most common way to style CSS is using an external CSS Style Sheet
  • We can link an external CSS style sheet using a "link" element, this links it to an external file
  • To apply a rule to all of the visible elements, we can use a wild card symbol or "*"
  • Extra notes: ref - refers to relationship between document and linked document source, href - refers to the location of the external resource, can be a URL

  • git checkout main: reverts to main branch
